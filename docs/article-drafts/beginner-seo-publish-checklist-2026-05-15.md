# Beginner SEO Publishing Checklist

Use this while reviewing drafts in `docs/article-drafts/beginner-seo-article-pack-2026-05-15.md`.

## Per-article checklist

- [ ] Title is clear and beginner-friendly (no jargon-heavy opener).
- [ ] Meta description is under ~160 characters and matches article intent.
- [ ] Teaser is short and useful.
- [ ] Article Type is correct (`beginnerCardPlay`, `beginnerDefence`, or `beginnerBidding`).
- [ ] Subcategory matches the allowed beginner subcategory list.
- [ ] Intro explains who this is for in plain English.
- [ ] Includes one practical checklist/process section.
- [ ] Ends with one clear takeaway.
- [ ] Links to one previous or foundational beginner article.
- [ ] Links to one next-step beginner article.

## Bulk publish run

- [ ] Publish first 6 articles in suggested order.
- [ ] Verify each published page has correct title and description in `<head>`.
- [ ] Verify each page appears in beginner hub list and opens correctly.
- [ ] Confirm "Next lesson" flow reads naturally between articles.
- [ ] Publish remaining 6 articles.
- [ ] Regenerate sitemap + prerender before deploy.

## Quick URL checks after publish

- [ ] `/beginner`
- [ ] `/beginner/articles/declarer`
- [ ] `/beginner/articles/defence`
